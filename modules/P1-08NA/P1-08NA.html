    P1-08NA

    8 Channel Discrete Input

    This module can return whether or not each of its eight channels are receiving voltage within a certain range.

    Here is a call of the readDiscrete function where 1 is the slot of the P1-08NA and 2 is the channel being read.

    P1.readDiscrete(1,2);

    Here is a call of the readDiscrete function where only the slot of the P1-08NA is given as an argument. This returns data where each channel is mapped to a specific bit starting with channel 1 as the least significant. This method is faster than when addressing channels individually.

    P1.readDiscrete(1);

    The above call would return 0xFF if all channels are on.

    Compatible Functions

    Below are functions compatible with the P1-08NA.

    • P1.readBlockData()

    Module Data

    This module does not provide any status data and does not require configuration.

    Discrete Input Type: AC
    Nominal Input Voltage: 120-240 VAC
    Example: DiscreteInput
