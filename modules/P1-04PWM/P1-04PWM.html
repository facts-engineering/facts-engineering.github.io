P1AM

    P1-04PWM

    Pulse Width Modulation Output

    This module can be used to generate up to four PWM signals for use in control applications.

    This code sets the PWM output signal to a 10kHz signal with a 50.15% duty cycle.

    float dutyCycle = 50.15; // Number between 0 and 100 that can have up to 2 decimal places
				int frequency = 10000;	// Frequency of signal in Hz
				P1.writePWM(dutyCycle, frequency, 1, 2); // Set duty cycle and frequency of slot 1 channel 2
