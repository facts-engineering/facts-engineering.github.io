P1AM

    P1-04NTC

    Thermistor Input

    This module can be used to get temperature readings from a number of different types of NTC probes. This function returns the temperature reported by the channel as a floating point value.

    float temperature = P1.readTemperature(1, 2); // Reads slot 1 channel 2 of the P1-04NTC
